# Useful Prompts for This Project

## Starting New Work

**Create issue from discussion:**
```
Based on our conversation about [topic], create a GitHub issue
following our template in .github/ISSUE_TEMPLATE/feature.md
```

**Work on an issue:**
```
Work on issue #N. Follow the workflow in .clinerules.
```

## Review Workflow

**Review in Cursor:**
```
Review the changes for issue #N against the acceptance criteria.
Use the checklist in .cursorrules.
```

**Check before committing:**
```
Before I commit these changes, verify:
- No secrets or personal data in committed files
- All acceptance criteria met
- README updated if needed
```

## Common Tasks

**Add new data source:**
```
Let's discuss adding [data source] to our aggregator.
What would we need to implement?
```

**Troubleshoot API issues:**
```
The [API] call is failing with [error]. Help me debug this.
Check the API docs and our error handling.
```

**Refactor existing code:**
```
The code in [file] works but is messy. Create an issue
for refactoring it to be clearer.
```

## When Context is Full

**Summarize progress:**
```
Summarize what we've accomplished in the last 5 issues.
Update PROGRESS.md with this summary.
```

**Start fresh:**
```
Read .clinerules and the open issues. What should we work on next?
```
